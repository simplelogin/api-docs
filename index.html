/auth

POST auth

Authenticate a user and returns the access_token.

Request URL:

https://api.simplelogin.co/v1/auth

Sample Response

{
  "status": "OK",
  "iterations": 5000,
  "token": "3243b4g5uy4fg1u3243o5uin23i4u123o4"
}

/users

Is a collection of user data

GET users/me

Return user data.

Request URL:

https://api.simplelogin.co/v1/users/me?user_id=[your_user_id]&access_token=[your_access_token]

Sample Response

{
  "status": "OK",
  "user": {
    "id": "51a280f23aa0f2c702000004",
    "username": "condorito",
    "first_login": true,
    "name": {
      "first_name": "René",
      "last_name": "Favaloro"
    },
    "email": "rene@simplelogin.co",
    "premium": true,

    "lang": "es",
    "gender": "M",
    "country": "ar",

    "billing_date": 123434234
  }
}

PUT users/me

Update user data.

Request URL:

https://api.simplelogin.co/v1/users/me?user_id=[your_user_id]&access_token=[your_access_token]

Sample Response

{
  "status": "OK",
  "user": {
    "id": "51a280f23aa0f2c702000004",
    "username": "condorito",
    "name": {
      "first_name": "René",
      "last_name": "Favaloro"
    },
    "email": "rene@simplelogin.co",
    "premium": true,
    "lang": "es",
    "gender": "M",
    "country": "ar"
  }
}

PUT users/me/first_login

Set first_login flag false.

Request URL:

https://api.simplelogin.co/v1/users/me/first_login?user_id=[your_user_id]&access_token=[your_access_token]

Sample Response

{
  "status": "OK"
}
